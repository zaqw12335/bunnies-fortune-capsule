# Bunnies Fortune Capsule

A fan-made bilingual React single-page interactive site.

## Local preview

```bash
npm install
npm run dev
```

Then open the local URL printed in the terminal.

## Build

```bash
npm run build
```

The production output will be in the `dist` folder.

## Deploy option A: Vercel

1. Create a new GitHub repository.
2. Upload this project folder.
3. Go to Vercel and import the repository.
4. Framework preset: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.

## Deploy option B: Netlify

1. Drag this project folder into a GitHub repository, or upload it directly through Netlify.
2. Build command: `npm run build`.
3. Publish directory: `dist`.
4. Deploy.

## Deploy option C: GitHub Pages

For GitHub Pages, either use a Vite GitHub Pages workflow or deploy the built `dist` folder through a Pages branch.
